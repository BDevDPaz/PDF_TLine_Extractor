🔧 Pila Tecnológica
  • Backend: Python 3, Flask
  • Base de Datos: SQLite con SQLAlchemy ORM
  • Procesamiento: Pandas, pdfplumber
  • Frontend: HTML5, TailwindCSS, JavaScript
  • Visualización: Chart.js, AG-Grid
  • IA: Google Gemini API
